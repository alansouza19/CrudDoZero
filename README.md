# CrudDoZero

Crud para listar e salvar usuários.

O usuário entra com os seguintes dados:

-Nome

-Data de Nascimento

-CEP

-Documento

Será retornado para o usuário os seguintes dados:

-Nome

-Idade

-CEP

-Documento

-Estado

-Cidade

-Bairro

O Serviço utiliza uma api para preencher os dados de endereço com o CEP:

https://viacep.com.br/
